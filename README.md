# **Movie list**

프론트의 덕목인 `List` 화면 처리 연습을 위한 흔하디 흔한 영화 리스트 및 검색 기능 구현

## **Stack**

- React
- Typescript
- Hooks
- Styled-components
- Axios
- Redux
- Redux-saga
- React-router
